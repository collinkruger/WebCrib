TL;DR

Add the following to a parent element.

            
.the-parent {
    -webkit-transform-style: preserve-3d;
    transform-style: preserve-3d;
}

Then add the following to a child element.

            
.the-thing-to-vertically-center {
  position: relative;
  top: 50%;
  transform: translateY(-50%);
}
